Border
HTML: 
<div class="content-box">content-box</div>
<div class="border-box">border-box</div>
CSS: 
div {
	border: 1px solid black;
	width: 100px;
	padding: 10px;
	margin: 10px;
}
.content-box {
	box-sizing: content-box;
}
.border-box {
	box-sizing: border-box;
}

Setting the width on a `content-box` results in the padding and borders being added to the width.

100 + 10*2 + 1*2 = 122

Setting the width on a `border-box` results in the padding and borders being included in that width.

100 - 10*2 - 1*2 = 78

Margin doesn't come into the equation for either, only padding, border and content.
